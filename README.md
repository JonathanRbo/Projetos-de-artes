# JR Design - Portfolio de Arte Digital

Portfolio pessoal de design gráfico com interface inspirada no Photoshop/Figma.

## Sobre

Página single-page que exibe amostras de trabalhos de design gráfico (artes para redes sociais, branding, peças publicitárias) através de embeds diretos do Instagram. O layout simula ferramentas de design com topbar e galeria central.

## Tecnologias

- **HTML5 / CSS3** — estrutura e estilização customizada
- **[Squeleton v4](https://squeleton.dev)** — framework CSS utilitário (grid, flexbox, animações)
- **WOW.js** — animações on-scroll
- **Google Fonts** — JetBrains Mono + Inter

## Estrutura

```
portfolio-artes/
├── index.html    # Página única com todo o conteúdo
└── README.md
```
