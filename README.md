# JR Design - Portfólio de Artes Digitais

Portfólio pessoal de design gráfico com interface inspirada no Photoshop/Figma.

## Sobre

Página single-page que exibe amostras de trabalhos de design gráfico (artes para redes sociais, branding, peças publicitárias) através de embeds diretos do Instagram. O layout simula ferramentas de design com topbar, painéis laterais (cores, camadas, informações) e galeria central.

## Tecnologias

- **HTML5 / CSS3** — Estrutura e estilização customizada
- **[Squeleton v4](https://squeleton.dev)** — Framework CSS utilitário (grid, flexbox, animações)
- **WOW.js** — Animações on-scroll
- **Google Fonts** — JetBrains Mono + Inter

## Estrutura

```
portfolio-artes/
├── index.html      # Página HTML (estrutura e conteúdo)
├── style.css       # Estilos CSS separados
├── favicon.svg     # Ícone da aba do navegador
├── og-image.svg    # Imagem de preview para redes sociais
└── README.md
```
